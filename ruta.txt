📅 Semana 1 - Introducción a JavaScript y Variables
🔥 Temas a estudiar:
✅ ¿Qué es JavaScript y para qué sirve?
✅ Cómo ejecutar JS en el navegador y en la consola.
✅ Variables: var, let, const y diferencias entre ellas.
✅ Tipos de datos: string, number, boolean, null, undefined, symbol.
✅ Operadores básicos: +, -, *, /, %, ++, --.
✅ Introducción a console.log(), typeof y alert().

📖 Recursos:
JavaScript.info - Variables
MDN - Tipos de Datos
Curso de YouTube - JS desde Cero (Fazt)
🎯 Ejercicios prácticos:
1️⃣ Crea una variable nombre y asígnale tu nombre. Luego usa console.log() para mostrarlo.
2️⃣ Declara una variable edad, suma 5 años a tu edad y muestra el resultado.
3️⃣ Usa typeof para mostrar el tipo de las variables anteriores.
4️⃣ Prueba con alert() para mostrar un mensaje de bienvenida.

📅 Semana 2 - Condicionales y Bucles
🔥 Temas a estudiar:
✅ Condicionales (if, else if, else, switch).
✅ Operadores de comparación: ==, ===, !=, !==, <, >, <=, >=.
✅ Operadores lógicos: &&, ||, !.
✅ Bucles (for, while, do while).
✅ Introducción a funciones (function, return, parámetros).

📖 Recursos:
JavaScript.info - Condicionales
MDN - Bucles e Iteración
JavaScript.info - Funciones
🎯 Ejercicios prácticos:
1️⃣ Escribe un programa que pida un número y diga si es par o impar.
2️⃣ Crea una función que reciba una edad y diga si la persona es mayor de edad.
3️⃣ Escribe un bucle for que imprima los números del 1 al 10.
4️⃣ Escribe un while que sume los números del 1 al 100.

📅 Semana 3 - Funciones y Arrays
🔥 Temas a estudiar:
✅ Funciones: declaración y expresión.
✅ Funciones de flecha (=>).
✅ Arrays y sus métodos básicos (push, pop, shift, unshift, length).
✅ Iterar sobre un array con for y forEach().

📖 Recursos:
JavaScript.info - Arrays

🎯 Ejercicios prácticos:

2️⃣ Crea una función que reciba un array y devuelva el mayor número.
3️⃣ Usa forEach() para recorrer un array de nombres e imprimirlos en consola.

✅ Eventos (addEventListener, click, keyup).
✅ Introducción a setTimeout() y setInterval().
